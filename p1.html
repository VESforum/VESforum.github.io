Theory:

HyperText Markup Language (HTML) is the standard markup language for creating web pages and web applications. With Cascading Style Sheets (CSS), and JavaScript, it forms a triad of cornerstone technologies for the World Wide Web. Web browsers receive HTML documents from a webserver or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document. HTML documents imply a structure of nested HTML elements. These are indicated in the document by HTML tags, enclosed in angle brackets. In the simple, general case, the extent of an element is indicated by a pair of tags: a "start tag" and "end tag". The text content of the element, if any, is placed between these tags. Tags may also enclose further tag markup between the start and end, including a mixture of tags and text. This indicates further (nested) elements, as children of the parent element. The start tag may also include attributes within the tag. These indicate other information, such as identifiers for sections within the document, identifiers used to bind style information to the presentation of the document. Some elements, such as the line break, do not permit any embedded content, either text or further tags. These require only a single empty tag (akin to a start tag) and do not use an end tag.

List of commonly used tags:

  1. Comment tag : Defines a comment
  2. a : Defines a hyperlink
  3. b : Defines bold text
  4. body : Defines the document's body
  5. br : Defines a single line break
  6. button : Defines a clickable button
  7. div : Defines a section in a document
  8. h1 to h6 : Defines HTML headings
  9. hr : Defines a thematic change in the content
  10. html : Defines the root of an HTML document
  11. img : Defines an image
  12. li : Defines a list item
  13. mark : Defines marked/highlighted text
  14. meta : Defines metadata about an HTML document
  15. nav : Defines navigation links
  16. ol : Defines an ordered list
  17. p : Defines a paragraph
  18. span : Defines a section in a document
  19. strong : Defines important text
  20. style : Defines style information for a document
  21. sub : Defines subscripted text
  22. sup : Defines superscripted text
  23. table : Defines a table
  24. title : Defines a title for the documentDefines a title for the document
  25. ul : Defines an unordered list
  26. var : Defines a variable
  27. video : Defines a video or movie
